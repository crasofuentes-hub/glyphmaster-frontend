# GlyphMaster Frontend

Frontend para el sistema GlyphMaster: generación de una fuente tipográfica a partir de caligrafía del usuario y composición en páginas tamaño carta (máx. 10 páginas).

## Objetivo
- Subida de muestras (imágenes) de escritura
- Envío al backend para entrenamiento/generación tipográfica
- Descarga/carga de fuente resultante (OTF/TTF/WOFF2)
- Editor que compone texto en páginas tamaño carta usando la fuente generada

## Seguridad
- La API key de Gemini no se almacena en este repositorio (se deja vacía en el frontend).
